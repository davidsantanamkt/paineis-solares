Dúvidas Frequentes

Respostas para suas principais questões

Quanto tempo leva para instalar? +
A instalação típica leva 3-5 dias, dependendo do tamanho do projeto. Após a instalação, leva 15-30 dias para aprovação da distribuidora e ativação do sistema.
Qual é a garantia dos painéis? +
Os painéis solares têm garantia de 25 anos de funcionamento. O inversor tem garantia de 10 anos. Oferecemos suporte técnico vitalício.
E se chover muito ou nevar? +
Os painéis funcionam mesmo com nuvens. A chuva até limpa os painéis naturalmente. Em regiões com neve, o sistema é projetado para suportar o peso.
Preciso fazer manutenção? +
Manutenção mínima. Limpeza anual dos painéis é recomendada. O sistema é monitorado remotamente 24/7 para detectar qualquer problema.
Como funciona a conexão com a rede? +
Seu sistema se conecta à rede elétrica. Quando você gera mais energia que consome, o excedente é enviado à rede e você recebe créditos na conta.
